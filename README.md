Config your vim from mine:
==============

Step 1
--------------
```shell
cd ~ && git clone https://github.com/wyying/.vim.git
```

Step 2
--------------
```shell
ln -sf ~/.vim/vimrc ~/.vimrc
```

Step 3
--------------
```shell
vim temp.txt
```

And then, type command below in the vim:
```shell
:PluginInstall!
```

Step 4
--------------
So easy! Rock your vim, now!!!
